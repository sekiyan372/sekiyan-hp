# Sekiyan's portfolio for Next.js

**Next.js + Tailwind CSS + Vercel**

## このサイトについて

このサイトは Sekiyan のポートフォリオサイト兼おもちゃです。
Sekiyan が気ままに作成・編集しています。

## 開発環境

### バージョン

- Node.js v18.12.0
- yarn 1.22.10
- typescript@3.9.3

### 環境変数

```
NEXT_PUBLIC_USER_ID
NEXT_PUBLIC_SERVICE_ID
NEXT_PUBLIC_TEMPLATE_ID
```
